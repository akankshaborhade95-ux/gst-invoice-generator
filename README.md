GST Invoice Generator

A simple GST Invoice Generator built using React.
It helps you create invoices, calculate tax, preview them, and print the invoice section.

Features:

1.Seller & Buyer details form
2.GSTIN, Email, Phone validation
3.Add multiple line items
4.Auto subtotal & GST calculation
5.CGST/SGST for same state
6.IGST for different state
7.Invoice preview
8.Print only invoice (not form)

 Run Project:
 1.Install packages:
   npm install
 2.Start app
   npm start (http://localhost:3000)
